# chicago-airbnb-data

## Description
chicago-airbnb-data is a project analyzing data from Chicago Airbnb listings and compiling it into a single report.

## Features
- Reads and processes Airbnb dataset files.
- Calculates information regarding listing types, hosts with multiple listings, and prices of rentals.
- Displays summary in an easy-to-read manner.

## Tech Stack
- Language: Python 3

## Motivation 
This project was created for a class project that focused on using Python for data analysis.

```bash
# Clone this repository
git clone https://github.com/diyaraj2028-code/chicago-airbnb-analysis

# Navigate into the project folder

# Run the script
